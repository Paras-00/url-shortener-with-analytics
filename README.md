**SyncVia** is a versatile web application that combines **URL shortening with analytics**, **real-time chat rooms**, and **shared video/lecture watching**. Built with **HTML, CSS, JavaScript, Node.js, and Express**, it provides an interactive experience where users can shorten links, track clicks, chat in real-time, and watch videos together.

## Features
- **URL Shortener & Analytics**
  - Shorten long URLs into easy-to-share links
  - Track analytics such as number of clicks and link creation date
  - Store link data in **local storage** or a database
  - Export analytics as PDF (optional)

- **Real-Time Chat Rooms**
  - Create or join chat rooms
  - Send and receive messages instantly
  - Share links or media with others

- **Shared Video Watching**
  - Watch lectures or MP4 videos together in real-time
  - Sync playback across all participants
  - Ideal for online study sessions or group video watching

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB / LocalStorage
- Libraries: jsPDF (for PDF export), Socket.io (real-time chat), PDF.js (optional), Video.js (for synchronized video playback)
